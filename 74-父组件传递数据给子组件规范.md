# 父组件（通过props传递数据）
```
<data-select 
  :min-width="'100px'" 
  :all-data-list="containerAllData"
></data-select>
```
# 子组件（在props接收）
```
export default {
  props: {
    // 其他已有的 props...
    allDataList: {
      type: Array,
      default: () => []
    }
  },
  ...
```
