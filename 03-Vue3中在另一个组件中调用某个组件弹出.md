## Vue3中在另一个组件中调用某个组件弹出
- 父组件
    - 调用子组件<Child ref="child" />
      - 通过ref可以拿到子组件的引用，可以获得子组件的实例，通过实例能够访问组件上定义的公开方法（methods）和数据（data）； 
    - 父组件可能通过某个点击对象来调用子组件
    ```
    const handleAdd = () => {
        addoreditDialog.value.open('add', {})； 父组件可能通过某个点击对象来调用子组件
    }
    ```
- 子组件
    - 书写父组件想要的open方法，
    - 然后通过expose暴露出去即可；暴露以后父组件就可以拿到了
