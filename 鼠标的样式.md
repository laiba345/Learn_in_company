# 内容
```
作为用户，我想要集成Genius系统中的设置项，以便于工程师进行系统设置。

【功能概要】

新增独立的设置中心

设置中心包含功能有：

程序版本管理（Package Version）

测试Cell 锁（Cell Lock）

测试终止权限管理（Cell Abort）

设置中心的权限为：engineer / engineer + admin

【操作流程】

进入设置中心

用户通过点击工号姓名的下拉框中的Settings

进入到System Settings页面

页面左侧显示导航，右侧默认显示Package Version

页面切换流程

 点击左侧导航的某个功能

右侧显示对应内容

【界面说明】

页面整体结构

 设置中心头部区域：页面标题：System Settings

左侧导航：

Package Version

Cell Lock

Cell Abort

右侧内容：显示所选中的页面内容

导航交互规则：

 当前选中项高亮显示

鼠标移入移出效果

点击后切换右侧内容

切换时页面不跳转，仅内容区域切换

原型地址：package version

【约束条件】

【详细说明】

右侧内容区域

左上角显示页面标题

页面标题下方显示页面说明

页面说明下方显示搜索项

搜索项输入后点击搜索样式即触发

搜索项下方显示列表

右上角显示功能按钮

Package Version

 针对Repository、Genius Version和TOP Version字段，显示当前版本

若当前版本不为最新版本，显示红色背景，鼠标悬浮显示该代码库的最新版本。否则显示绿色
```
